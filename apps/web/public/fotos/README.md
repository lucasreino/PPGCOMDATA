# Fotos dos docentes

Coloque aqui as imagens com um destes padrões de nome:

- `{id_lattes}.jpg` (recomendado — ex.: `1234567890123456.jpg`)
- `{id_do_professor_no_sistema}.jpg` (UUID retornado pela API)
- `.png` ou `.webp` também funcionam

Alternativa: defina `foto_url` no cadastro do professor (caminho `/fotos/arquivo.jpg` ou URL externa).

Sem arquivo, o sistema exibe avatar com as iniciais do nome.
